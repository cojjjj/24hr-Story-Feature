# 📸 24hr Story Feature

A modern frontend-only Story feature inspired by Instagram and WhatsApp that allows users to upload, view, and manage temporary stories that automatically disappear after 24 hours.

---

## ✨ Features

- 📷 Upload images as stories
- 💾 Store stories locally using browser Local Storage
- 🔄 Convert uploaded images to Base64 format
- ⏰ Automatic 24-hour story expiration
- 👆 Click to view stories in fullscreen mode
- ➡️ Navigate between stories
- 📱 Fully responsive design
- 🚀 Client-side only implementation


Navigate to the project directory:

```bash
cd stories-project
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## 🧠 How It Works

1. User selects an image using the **+** button.
2. The image is converted to **Base64** using the FileReader API.
3. Story data and timestamp are saved in **Local Storage**.
4. Stories are displayed in the story carousel.
5. Users can open stories in a fullscreen viewer.
6. Expired stories are automatically removed after 24 hours.
